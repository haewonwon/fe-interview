# 📂 Algorithm 개요

## 📌 학습 목표

알고리즘은 **문제 해결 능력을 향상시키고, 효율적인 코드 작성의 핵심 기반**입니다.  
이 레포지토리는 다음과 같은 목적을 가지고 학습 내용을 정리합니다:

- 🔹 **기본 알고리즘 유형 정리** (정렬, 탐색, 그리디 등)
- 🔹 **문제 해결 전략 및 패턴 학습** (투포인터, 백트래킹 등)
- 🔹 **코딩 테스트 및 면접 대비**
