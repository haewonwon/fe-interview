# FE Interview 📚

이 레포지토리는 **프론트엔드 면접을 대비하기 위해 공부한 것을 기록**하는 공간입니다.  
CS뿐 아니라 프론트엔드 전반에 걸친 주요 개념들을 정리하여, 면접 대비 및 지식 정리에 활용할 수 있도록 구성했습니다.

## 📂 폴더 구조 및 내용

### 🖥️ Frontend
- [**core/**](frontend/core/) — 브라우저 렌더링, 네트워크, 보안 등 CS ↔ 프론트엔드 연결 개념
- [**language/javascript/**](frontend/language/javascript/) — JavaScript 핵심 개념
- [**language/typescript/**](frontend/language/typescript/) — TypeScript 핵심 개념
- [**library-framework/react/**](frontend/library-framework/react/) — React
- [**library-framework/next-js/**](frontend/library-framework/next-js/) — Next.js
- [**library-framework/react-native/**](frontend/library-framework/react-native/) — React Native / WebView
- [**backend-basic/**](frontend/backend-basic/) — BFF, REST API, Node.js 등
- [**ecosystem/**](frontend/ecosystem/) — 패키지 매니저, 번들러, 테스팅, CI/CD
- [**portfolio/**](frontend/portfolio/) — 프로젝트별 기술 선택 이유 정리

### 💻 CS
- [**cs/algorithm/**](cs/algorithm/) — 정렬, 탐색, 분할 정복 등
- [**cs/data-structure/**](cs/data-structure/) — 배열, 스택, 큐, 트리, 그래프 등
- [**cs/computer-networks/**](cs/computer-networks/) — 네트워크 기초, HTTP
- [**cs/operating-system/**](cs/operating-system/) — 프로세스, 스레드, 커널 등
- [**cs/database/**](cs/database/) — DB 기초
- [**cs/computer-architecture/**](cs/computer-architecture/) — 컴퓨터 구조 기초
