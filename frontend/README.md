# 🖥️ Frontend 개요

## 📌 학습 목표

프론트엔드 개발은 **사용자와 직접 맞닿는 화면을 구현하고 인터랙션을 다루는 핵심 영역**입니다.  
이 레포지토리는 다음과 같은 목적을 가지고 학습 내용을 정리합니다:

- 🔹 **웹 기본 동작 원리 이해** (HTML/CSS/JS 기반 구조 및 흐름)
- 🔹 **프레임워크와 상태 관리 학습** (React, Zustand, Jotai 등)
- 🔹 **면접 대비** (자주 나오는 프론트엔드 핵심 질문 대비)
