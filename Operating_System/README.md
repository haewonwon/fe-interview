# 💻 Operating System 개요

## 📌 학습 목표

운영체제는 **하드웨어 자원을 효율적으로 관리하고, 응용 프로그램에게 서비스를 제공하는 핵심 시스템 소프트웨어**입니다.  
이 레포지토리는 다음과 같은 목적을 가지고 학습 내용을 정리합니다:

- 🔹 **핵심 운영체제 개념 정리** (프로세스, 스레드, 스케줄링, 메모리 관리 등)
- 🔹 **시스템 자원 관리 및 동작 원리 이해** (커널, 시스템 콜, 동기화)
- 🔹 **면접 대비 및 CS 전공 지식 확립**
